# Codex skills 2.0.0 (2026-02-06)

## Highlights
- Standardize skill installs to the new `.agents/skills` catalog (`~/.agents/skills` by default, or `--dir .agents/skills` for repo-local installs).
- Simplify the CLI: remove multi-agent installs and replace `--agent` with `--dir`.
- Update the skill catalog: add `ui-ux-pro-max`; remove `frontend-design`, `brave-search`, and `oracle`.

## Breaking changes
- CLI: removed `--agent` / multi-agent install targets. Use `--dir <dir>` instead.
- CLI: default install dir changed from `~/.codex/skills/` to `~/.agents/skills/`.
- Skills removed from the catalog: `frontend-design`, `brave-search`, `oracle`.

## Upgrade notes
- If you previously installed skills into `~/.codex/skills/`, reinstall into the new default:
  - `npx codex-skills install-all`
- To install into a repo (portable), use:
  - `npx codex-skills install-all --dir .agents/skills`
- If you added scripts to PATH, update it to the new location:
  - macOS/Linux: `export PATH="$PATH:$HOME/.agents/skills/agent-scripts"`
  - Windows (PowerShell): add `$HOME\\.agents\\skills\\agent-scripts` to PATH
